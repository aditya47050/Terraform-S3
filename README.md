# AWS S3 Static Website using Terraform

This project demonstrates how to deploy a **static website on AWS S3** using **Terraform (Infrastructure as Code)**.  
The website is publicly accessible via S3 website endpoints and the entire infrastructure lifecycle is managed using Terraform.
